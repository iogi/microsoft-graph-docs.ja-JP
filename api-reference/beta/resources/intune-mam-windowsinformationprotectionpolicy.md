|minutesOfInactivityBeforeDeviceLock|Int32|デバイスがアイドルになった後に、デバイスの PIN またはパスワードがロックされるまでの最大時間 (分) を指定します。   範囲は整数 X (0 < = X < = 999) です。|
|daysWithoutContactBeforeUnenroll|Int32|アプリのデータがワイプされるまでのオフライン期間 (日数) |
